# دليلي الشخصي — شرح كل حاجة في المشروع ## مقدمة الملف ده خاص بيا أنا — بيشرح كل قرار اتخذته في المشروع، ليه اتعمل، وفين بالظبط في الكود. --- ## 1. EcommerceSharedDataService **الملف:** `app/Services/EcommerceSharedDataService.php` **ليه موجود؟** كل صفحة في الموقع محتاجة نفس البيانات: اسم الفرع، التليفون، اللوجو، لينكات الـ navbar، التصنيفات. لو كتبت نفس الكود في كل Controller هيبقى "code duplication" وصعب التعديل. الحل: `Service` واحدة بتجيب كل البيانات دي مرة واحدة. **الاستخدام في أي Controller:** ```php $ecommerceSharedData = EcommerceSharedDataService::get(); ``` **الـ return value:** ```php [ 'branchName' => 'اسم الفرع', 'branchImage' => 'صورة الفرع', 'phone' => 'رقم التليفون', 'logo' => 'مسار اللوجو', 'navData' => ['categories' => [...], 'products' => [...]], 'staticLinks' => [...], // لينكات الـ navbar 'footerCategories' => [...], 'branch' => $Branch, // Object كامل من الـ DB 'social' => $Social, // روابط السوشيال ميديا 'mapUrl' => '#', // رابط الخريطة ] ``` --- ## 2. نظام السلة — Session-based Cart **الملفات:** - `app/Http/Controllers/Ecommerce/ShoppingCart/ShoppingCartController.php` - `resources/views/ecommerce/ShoppingCart/ShoppingCart.blade.php` **الفكرة الأساسية:** السلة محفوظة في `session('cart')` — يعني في الـ Server، مش في الـ Browser. لما المستخدم بيضيف منتج، بيبعت AJAX request للـ Controller، الـ Controller بيعدل الـ session، وبيرجع JSON. **شكل الـ Session:** ```php // session('cart') بيكون array زي ده [ '123' => [ // المفتاح = ecommerceproduct_id 'id' => '123', 'name' => 'آيفون 15', 'price' => 35000, 'offer_price' => 33000, 'image' => 'iphone15.jpg', 'quantity' => 2, ], '456' => [...], ] ``` **ليه session ومش DB؟** لأن السلة محتاجة تشتغل بدون تسجيل دخول. لما المستخدم يسجل دخول ويضغط "متابعة الشراء"، `CheckoutController::confirm()` بيحول الـ session للـ DB في جدول `customerrequestproducts`. **الحدود (Limits):** - `CART_MAX_ITEMS = 20` — أقصى عدد منتجات مختلفة - `CART_MAX_QTY = 10` — أقصى كمية لكل منتج الحد بيتطبق في مكانين: 1. **JavaScript** (في `layouts/app.blade.php`) — بيمنع الـ request قبل ما يوصل للـ Server 2. **Controller** (في `CustomerRequestIncreseQuantityPost`) — الحماية الحقيقية --- ## 3. الـ AJAX — ليه مش بنعمل Refresh؟ **الملف:** `resources/views/layouts/app.blade.php` — في الـ `
